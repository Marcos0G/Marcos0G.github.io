Modelo de caja

Se trata de una caja que envuelve los elementos de html. Consta de márgenes, bordes, relleno y el contenido real. Utiliza etiquetas como padding o border (que define los bordes de la caja)

por ejemplo podemos ponerle un marco a este texto y "encerrarlo" en una caja en este caso tendría un marco de 10 px con la parte de arriba mas oscura y un margen de 5%

Posicionamiento (relativo, absoluto, arreglado y z-index)

-Relativo: coloca el elemento donde lo haría normalmente pero compensando un poco, esta compensación se determina con 4 nuevas etiquetas:
Top, bottom, right y left. Se despliega bajo todo. -Absoluto: se saca al elemento del "flujo normal" para colocarlo en cualquier parte de la pantalla (el escenario y la criatura serián un ejemplo) -arreglado/fixed: hace parecer que las cosas no se mueven dejandolas fijas en la pantalla (un ejemplo de esto es el título de esta página) -Z-index: esta propiedad se usa para indicar al navegador el orden en el que tiene que dibujar los elementos, dandoles diferentes indices
(z-index 1, z-index 2, ...) estos elementos de debajo tienen z-index aplicado.

Elementos flotantes

Se usan para que los elementos queden "flotando" y el texto se acomode a su alrededor.
Por ejemplo acomodamos este texto a la foto flotando a laizquierda.


También tendriamos el atributo clear, usado para que se acomode donde queramos. puede ser both (ira abajo), o right/left si no lo queremos solo a un lado.
Por ejemplo el "pie de pagina" tiene clear both.

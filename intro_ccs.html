Seleccion a partir de una etiqueta

Podemos cambiar las característucas del texto usando las etiqutas,
para hacerlo simplemente debemos usar las etiqutas style y dentro
escribir el nombre de las etiquetas que queremos cambiar seguidas de {} y elegir
el color. Los títulos tienen el color de las letras y el fondo también están cambiados
usando esta selección.

Selección a partir de id (#)

Se pueden usar id en las etiquetas para seleccionar algunas especificas en vez de todas
como haría la simple elección por etiqueta. Para poder hacer esto, lo primero que tenemos
que hacer es ir a la etiqueta que queremos seleccionar y escribir id= y el nombre del id que
vamos a usar entre " (ejemplo: p id=""). Después, volveremos a las etiquetas style y pondremos
el # y el nombre del id y lo demas sera seleccionar el color como con las etiquetas normales.

Selección por clase

No se puede usar el mismo id para diferentes elementos, tiene que ser único.Para seleccionar
varios elementos usando el id, deberíamos ir inventando uno nuevo para cada cosa. Para evitar
esto esta la etiqueta class. Esta sirve para agrupar varios elementos a un mismo grupo y darles
características específicas. Para usar class debemos ir a las etiquetas que vayamos a cambiar
y dentro de estas escribir class="" (ejemplo: p class=""). Luego volvemos a style y escribimos
. y el nombre del id seguido (.explicación por ejemplo). Este parrafo esta seleccionado con class
